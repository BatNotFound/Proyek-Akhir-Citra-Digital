# Proyek-Akhir-Citra-Digital
Repository ini berisi dataset_wajah yang digunakan untuk proyek akhir mata kuliah TK1: Citra Digital mengenai _face recognition_ menggunakan pre-trained model ResNet50.

Dataset ini merupakan subdatset dari dataset publik _PINS Face Recognition_ yang dapat diakses melalui repositori Kaggle dari link berikut: https://www.kaggle.com/datasets/hereisburak/pins-face-recognition

Dari 105 selebritas, dipilih 11 selebritas sebagai data citra wajah untuk melatih model _face recognition_.
